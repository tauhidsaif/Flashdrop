How It Works

FlashDrop uses WebRTC and Socket.IO to transfer files directly between two devices on the same Wi-Fi or LAN. Nothing is stored on any server.

  1. Sender: Click "Send" and get a secure 6-digit code.
  2. Receiver: Enter that code to connect to the sender.
  3. File Transfer: Begins instantly via direct peer-to-peer connection using WebRTC.
